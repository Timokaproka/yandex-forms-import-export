# Yandex Forms Migrator (Экспорт/Импорт)

## Суть работы
Официальный редактор Яндекс Форм не имеет встроенной кнопки для переноса структуры формы (вопросов, настроек) из одного аккаунта в другой или из обычного Яндекса в Yandex Cloud. 

Это расширение добавляет возможность в один клик:
* Сделать **бэкап (экспорт)** всех вопросов формы в удобный JSON-файл.
* **Импортировать** готовую структуру из JSON-файла в любую пустую форму, экономя часы ручного переноса вопросов.

⚠️ **Обратите внимание:** Расширение переносит *только структуру вопросов* (названия, типы, варианты ответов, обязательность). **Настройки самой формы** (интеграции, условия показа, внешний вид, доступы, почтовые уведомления) **НЕ переносятся**. Их необходимо будет настроить вручную на новой форме!

## Инструкция по работе

### Экспорт в JSON
1. Зайдите в редактор нужной формы (ссылка вида `https://forms.yandex.ru/admin/*/edit`, где `*` - ID формы).
2. Нажмите на иконку расширения и выберите **«Экспортировать JSON»**. 
3. На ваш компьютер будет загружен `.json` файл, содержащий все данные о вопросах этой формы.

### Импорт из JSON
1. Создайте новую (пустую) форму и зайдите в её редактор (ссылка вида `https://forms.yandex.ru/admin/*/edit`).
2. Нажмите на иконку расширения и выберите **«Импортировать из файла»**. 
3. Выберите `.json` файл, который был ранее создан расширением.
4. Подождите окончания процесса (вопросы добавляются с небольшой задержкой для защиты от блокировок Яндекса).
5. После загрузки высветится окно о том, что импорт успешно завершён. 
6. **Перезагрузите страницу**, чтобы увидеть добавленные вопросы.
7. *Не забудьте вручную выставить нужные настройки формы (интеграции, логику, дизайн).*

## Скачивание
Перейдите в раздел [Releases](https://github.com/Timokaproka/yandex-forms-import-export/releases) и скачайте последнюю версию из архива `Forms_extension.zip`.

## Установка в браузере
*(Поддерживаются только браузеры на базе Chromium: Google Chrome, Яндекс Браузер, Edge, Opera и др.)*

1. Распакуйте скачанный `.zip` архив в любую удобную папку на компьютере.
2. Откройте в браузере страницу управления расширениями: `chrome://extensions/` (или `browser://extensions/` для Я.Браузера).
3. Включите **«Режим разработчика»** (обычно тумблер находится в правом верхнем углу).
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked).
5. В открывшемся окне выберите папку, в которую вы распаковали архив на шаге 1. Готово!
